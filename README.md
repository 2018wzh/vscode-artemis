# Artemis Engine Support for VSCode

This extension provides syntax highlighting for Artemis Engine `.iet` files in Visual Studio Code.

## Features

*   **Syntax Highlighting**: Comprehensive syntax highlighting for `.iet` script files used in the Artemis Engine.
*   **Language Support**: Automatically recognizes `.iet` file extensions.

## Requirements

No special requirements or dependencies.

## Extension Settings

This extension does not contribute any settings.
